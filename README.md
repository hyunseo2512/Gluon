# Gluon - AI 기반 IDE

> AI 통합 모던 개발 환경

Gluon은 로컬 LLM 엔진과 통합된 IDE입니다. 실시간 AI 채팅, 코드 분석, 스마트 터미널 등을 지원합니다.

---

## 주요 기능

- **AI 채팅**: DeepSeek, Llama 3 등 로컬 AI 모델과 실시간 대화
- **코드 에디터**: Monaco Editor 기반, 구문 강조, 자동 완성, 코드 포맷팅
- **분할 편집기**: 화면을 좌우로 분할하여 동시에 여러 파일 편집
- **Diff 보기**: Git 변경 사항 및 AI 코드 리뷰 결과를 시각적으로 비교
- **통합 터미널**: pty 기반 쉘 환경, 다중 탭 지원
- **파일 탐색기**: 생성/수정/삭제, 드래그 앤 드롭, 파일 아이콘
- **Git 통합**: 변경 확인, 스테이징, 커밋
- **SSH 원격 접속**: 원격 서버 파일 직접 편집
- **명령 팔레트**: 모든 기능 빠른 검색 및 실행

## 설치 및 실행

### 필요 조건
- Node.js 18+
- Python 3.10+ (AI 백엔드용)

### 설치
```bash
npm install
```

### 개발 모드
```bash
npm run dev
```

### 빌드 및 패키징 (Linux)
```bash
npm run build
npm run package    # 출력: release/
```

## 사용법

### 프로젝트 열기
- 실행 후 **폴더 열기** 버튼을 클릭하여 프로젝트 폴더를 선택합니다.

### 파일 편집
- 좌측 **파일 탐색기**에서 파일을 클릭하면 에디터에 열립니다.
- 탭을 드래그하여 순서를 변경할 수 있습니다.
- `Ctrl + S`로 저장합니다.

### 분할 편집기
- 탭 바 우측의 **분할 토글 버튼**을 클릭하면 화면이 좌우로 분할됩니다.
- 다시 클릭하면 단일 화면으로 돌아갑니다.

### AI 채팅
- 우측 **AI 패널**에서 현재 파일에 대한 질문이나 코드 리뷰를 요청할 수 있습니다.
- 파일 문맥을 자동으로 인식하여 답변합니다.

### 터미널
- `` Ctrl + ` ``로 터미널 패널을 열고 닫습니다.
- 여러 터미널 탭을 동시에 사용할 수 있습니다.

### Git
- 좌측 **Git 패널**에서 변경 사항을 확인하고 스테이징/커밋합니다.
- 파일을 클릭하면 Diff 보기로 변경 내용을 비교할 수 있습니다.

## 단축키

| 단축키 | 기능 |
|--------|------|
| `Ctrl + S` | 저장 |
| `Ctrl + W` | 현재 탭 닫기 |
| `` Ctrl + ` `` | 터미널 열기/닫기 |
| `Ctrl + ,` | 설정 |
| `Ctrl + Shift + P` | 명령 팔레트 |
| `Ctrl + \` | 분할 편집기 토글 |
| `Ctrl + Arrows` | 분할 창 간 이동 |
