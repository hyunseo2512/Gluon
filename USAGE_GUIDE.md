## 🚀 빠른 시작

### 1. 의존성 설치 및 실행

```bash
# Gluon 디렉토리로 이동
cd /home/zxcvne/ai-zagreus/Gluon

# nvm 활성화 (필요시)
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"

# 의존성 설치 (이미 완료)
npm install

# 개발 모드 실행
npm run dev
```

### 2. ZAGREUS 서버 실행 (선택사항)

AI 기능을 사용하려면 별도 터미널에서:

```bash
cd /home/zxcvne/ai-zagreus/ZAGREUS
source .venv/bin/activate
python run.py serve
```

## 📁 파일 탐색 및 편집

### 1. 폴더 열기

**방법 1: 버튼 사용**
1. 왼쪽 액티비티 바에서 📁 클릭
2. "폴더 열기" 버튼 클릭
3. 시스템 다이얼로그에서 원하는 폴더 선택

**방법 2: 헤더 버튼**
1. 탐색기 헤더의 📂 버튼 클릭
2. 폴더 선택

### 2. 파일 트리 탐색

- **📁 폴더 아이콘:** 클릭하면 하위 항목 펼치기/접기
- **📄 파일 아이콘:** 클릭하면 에디터에서 열기
- **정렬 순서:** 폴더 먼저 → 파일, 각각 알파벳순

### 3. 파일 편집 및 저장

1. 파일 클릭하여 에디터에서 열기
2. 내용 수정
3. **Ctrl+S** 눌러서 저장
4. 콘솔에서 저장 성공/실패 확인 가능

**저장 확인:**
- 브라우저 DevTools 열기 (개발 모드에서 자동 열림)
- Console 탭에서 `✅ 파일 저장 성공: /path/to/file` 확인

## 🔧 문제 해결

### Node.js가 없다고 나옴

```bash
# nvm 로드
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"

# Node.js 버전 확인
node --version
npm --version
```

### 폴더를 열 수 없음

- Electron dialog API가 제대로 로드되었는지 확인
- 개발자 도구 (F12)에서 에러 확인
- 폴더에 대한 읽기 권한 확인

### 파일을 저장할 수 없음

- 파일에 대한 쓰기 권한 확인
- 디스크 공간 확인
- 콘솔에서 에러 메시지 확인

### AI 패널이 작동하지 않음

1. ZAGREUS 서버가 실행 중인지 확인:
   ```bash
   curl http://localhost:8000/health
   ```
2. CORS 설정 확인 (ZAGREUS에 추가 필요)
3. 네트워크 에러 확인 (개발자 도구)

### 터미널 입력이 안 됨

- 터미널 영역 클릭하여 포커스
- 시뮬레이션 모드이므로 제한된 명령어만 사용 가능
